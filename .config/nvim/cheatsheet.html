New here? Start with these 4 things

1

The Leader key is Space. Most commands begin with it. Press it and pause — which-key pops up a menu of what comes next.

2

Three core modes: Normal (navigate), press i for Insert (type), v for Visual (select). Get back to Normal with Esc or jk.

3

Open files fast with Spaceff and search text with Spacefs. Toggle the file tree with Spaceee.

4

Files auto-format on save. Run the current script with SpaceR, and debug it starting with F5.
